🧠 Multi-Agent AI Deep Researcher

LangGraph + DeepSeek V3 + Tavily — Hackathon Architecture

👤 User Input
Research topic entered via terminal
AGENT 1
🗺️ Query Planner
Breaks the topic into 4 targeted sub-questions covering different research angles
DeepSeek V3 LangChain Prompts
→ Outputs: List of 4 sub-questions
AGENT 2
🌐 Contextual Retriever
Searches the live web for each sub-question — fetches titles, URLs, and content snippets
Tavily Search API Live Web
→ Outputs: 12 real web sources with content
AGENT 3
🔬 Critical Analyst
Summarizes each source, scores credibility, and flags contradictions between sources
DeepSeek V3 LangChain Prompts
→ Outputs: Summaries + contradiction analysis
AGENT 4
💡 Insight Generator
Reasons across all findings to surface trends, hypotheses, implications and open questions
DeepSeek V3 Chain of Thought
→ Outputs: Trends, hypotheses, implications
AGENT 5
📝 Report Builder
Compiles everything into a structured markdown report with executive summary and citations
DeepSeek V3 Markdown Output
→ Outputs: Full research report saved as .md file
📄 Final Research Report
Executive Summary · Findings · Contradictions · Insights · Sources
🐍 Python 3.9
🔗 LangGraph — Agent Orchestration
🧠 DeepSeek V3 via OpenRouter
🔍 Tavily — Live Web Search
⛓️ LangChain — Prompt Management
